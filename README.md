# springboot-project
This is a Spring Boot REST API for JounalApp. 
It supports CRUD operations like create, update, delete, and fetch users.

## Features
- Create user
- Get all users
- Update user
- Delete user
- RESTful APIs

## Tech Stack
- Java 17
- Spring Boot
- Maven
- MySQL

src/main/java/com/example/demo
├── controller
├── service
├── repository
└── model

## Prerequisites
- Java 17 installed
- Maven installed
- MySQL running
